Weather app

Get live weather data, store and view it in a simple and easy way.

Features about

Main Features

Live weather data

Fetch live weather data from OpenWeather API and present it to users in real time.

Data storage

Store weather data in a database for later access.

Show data

Display weather data in an easy-to-understand manner, with live updates when needed.

around

This app is developed to provide weather information directly and easily. We used OpenWeather API to fetch the data and stored it in a database to provide accurate and up-to-date information.
